The Platform
Different doors. Same intelligence.
DMP is a unified marketing operating system for Walt Disney Studios Motion Pictures. Every team gets a purpose-built interface into the same data layer — the full global slate, live title context, and tools designed for how they actually work.
Architecture

One brain.
Many front doors.

Every tool in DMP shares the same foundational data — the complete Disney release slate, title context, campaign timelines, and team structures. What changes is the interface: each specialty team enters through a door designed for their workflow, not a generic platform they have to adapt.

🧠
Unified slate data
All 40+ titles, 2026–2031, shared across every tool. Switch any title in any tool and the whole system recontextualizes.
🚪
Specialty front doors
PGS for publicity. Creative Director for creative. Each tool is purpose-built, not repurposed. More doors being built now.
⚡
Conversational interface
No forms. No dashboards to learn. Every tool is a conversation — describe what you need, get what you need.
📊
Logged & traceable
Google Sheets integration logs every action across all tools. Full audit trail, cross-team visibility.
